# SEO 최적화 가이드

## OG 이미지 추가 필요

`og-image.jpg` 파일을 이 폴더(public)에 추가해주세요.

### 권장 사양:
- **크기**: 1200 x 630px (Facebook/Twitter 권장)
- **포맷**: JPG 또는 PNG
- **파일명**: og-image.jpg
- **내용**: 서비스 로고 + "무료 증명사진 만들기" 텍스트

### 이미지 생성 도구:
- Canva: https://www.canva.com/
- Figma: https://www.figma.com/
- Photoshop 또는 기타 이미지 편집 도구

## 파비콘 추가

`favicon.ico` 파일도 추가해주세요.

### 권장 사양:
- **크기**: 32x32px, 16x16px (다중 크기 포함)
- **포맷**: ICO
- **파일명**: favicon.ico

## 네이버 검색 등록

1. 네이버 서치어드바이저 방문: https://searchadvisor.naver.com/
2. 사이트 등록 및 소유권 확인
3. 사이트맵 제출: https://letstest.cloud/sitemap.xml
4. 색인 요청

## 구글 검색 등록

1. Google Search Console 방문: https://search.google.com/search-console
2. 속성 추가 (이미 완료된 것으로 보임 - google-site-verification 태그 존재)
3. 사이트맵 제출: https://letstest.cloud/sitemap.xml
4. URL 검사 도구로 색인 요청

## 주요 타겟 키워드

1. **증명사진 만들기** (메인 키워드)
2. **여권사진 규격**

4. **이력서사진 사이즈**
